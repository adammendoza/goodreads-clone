<div align="center">
  <h2>Installation</h2>
</div>


### Clone

- Clone this repo to your local machine using `https://github.com/tjtaylorjr/goodreads-clone.git`

<br>

### Setup

> install npm packages

```shell
$ npm install
```

> refer to internal documentation for setting up database
<br>

> run the server

```shell
$ npm start
```
> browse to http://localhost:8080
